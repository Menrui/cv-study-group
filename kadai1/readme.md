# 課題１

## 関数の説明
* bilinier(bilinier.cpp)
  * バイリニア補完を掛ける際の、座標ごとの補完値を計算する。
  * 入力
    * 補完対象画像 Mat image
    * x座標
    * y座標
  * 出力
    * 与えられた座標のバイリニア補完値
* nearest(nearest.cpp)
  * ニアレスト補完を掛ける際の、座標ごとの補完値を計算する。
  * 入力、出力ともにbilinierと同じ。
* pixelAffine(affine.cpp)
  * アフィン変換を行う際の、座標ごとの変換値を計算する。アフィン変換後の座標からアフィン変換前の座標を計算する。
  * 入力
    * 変換行列 A
    * x座標
    * y座標
  * 出力
    * アフィン変換前の座標(x, y)を要素とするタプル
